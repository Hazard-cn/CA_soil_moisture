## 禁止的直接拼接

不能把G185的46,299行样本、事件方向的固定32°C热干事件和阈值方向的空间连续阈值共同称为一个冻结设计。三者在数据家族、样本谓词、暴露单位、降水条件、持续时间条件、SR端点和estimand上均不相同。任何跨暴露表征的新论文都必须把所有暴露重新构造到同一个V3共同样本、同一窗口、同一SR端点、同一FE和同一推断框架，并登记新canonical ID。

## 可行的先行SM新方向

较严谨的状态依赖设计应固定在V3事件家族中：以现有32°C、1 mm、3天规则识别热干事件，以SR连续值及共同P25/P75作图，不在确认性主方程中加入外生阈值，也不锁定G185样本。先行SM必须在看到产量结果前，从以下互斥方案中冻结一个，不能在结果生成后切换：

1. 若研究对象包含有事件和无事件grid-years，则对所有行使用同一固定物候风险窗口开始前14日至前1日GLEAM SMrz，不能只对事件行定义onset前SM。
2. 若研究对象限定为事件阳性grid-years，则以当季第一次合格事件的onset前14日至前1日GLEAM SMrz定义先行状态，estimand只解释事件阳性样本内严重程度的状态依赖，不解释事件发生边际。
3. 若需要利用一季内多个事件，则必须预先固定唯一聚合规则，例如第一次事件、最严重事件或事件级SM结果的grid-year等权汇总；产量方程仍保持每个`grid_id year`一行，不得把同一季产量复制为多个独立事件观测。

该方向应先作为修正后事件稿的预设次级模块，是否独立成稿取决于支持域、空间推断和新增信息。当前材料尚未冻结上述三种SM时序方案中的哪一种，因此网页端提出的S1或类似方案只能作为新canonical设计建议，不能被表述为现有确认性分析。
