# 条件斜率结果的统一分类规则

任何“高SR减低SR”的差值都必须与低、高SR两个端点的条件变化一起报告。分类分为两层：第一层只描述点估计的形态；第二层才判断该形态是否得到同一主推断下的统计确认。只有低SR端点、高SR端点及两者差值都使用同一主推断、区间满足相应符号条件且差值区间不跨0时，才可写为“统计确认的某类关系”。任一端点或差值区间跨0时，只能写“点估计呈现某种形态，但未被精确确认”。

| 低SR点估计 | 高SR点估计 | 高减低差值 | 点估计形态 |
|---|---|---|---|
| 负 | 负，且高SR更接近0 | 正 | 损失减弱形态 |
| 负 | 正 | 正 | 负向到正向的跨零反转形态 |
| 正 | 正，且高SR更高 | 正 | 正向条件斜率增强形态 |
| 正 | 正，且高SR更接近0 | 负 | 正向条件斜率减弱形态，不称损失缓冲 |
| 正 | 负 | 负 | 正向到负向的跨零反转形态 |
| 负 | 负，且高SR更负 | 负 | 负向恶化形态 |
| 任一端点或差值区间不满足确认条件 | 任意 | 任意 | 形态未被精确确认 |

G185的year FE全国P90端点中，干旱点估计为−2.113%→−1.548%，呈损失减弱形态；高温为−1.417%→−0.276%，也呈损失减弱形态，但高SR端点区间跨0；热干为+0.174%→+1.322%，呈正向增强形态，两个端点区间均跨0。province×year FE下，干旱为−0.935%→−1.224%，呈负向恶化形态；高温为−3.250%→−2.719%，热干为−3.857%→−3.406%，均呈损失减弱形态，但三个胁迫的端点区间均跨0，因此这些均是点估计分类，不是已确认的损失缓冲或恶化。

事件稿黄淮海季节累计事件天数的点估计为−3.34%→+7.71%，呈跨零反转形态；低SR端点区间跨0，不能写成统计确认的跨零反转。阈值稿黄淮海完整窗口两端点估计均为正，呈正向增强形态；由于端点区间和差值区间来自不同推断层级，当前材料也不能把该形态写成统一空间推断下的统计确认。

这些分类只描述模型中的条件关联，不把SR水平当作随机处理，不产生ATE、ACME、自然直接效应或自然间接效应。
