# 外生玉米阈值资产与算法对应说明

用户提供的官方`maize.tif`经本项目核验：MD5为`0d9e6c21bf1b25f113e14315863372f2`，SHA-256为`f05e634664c4c6c2e2df352702acd421507162bc54cc067649071521ab1285b0`。该文件是0.5°连续玉米温度阈值栅格；主映射按项目0.1°格网中心所在0.5°单元赋值，不对阈值四舍五入。

rank 1公开`CalExposure.m`计算逐小时温度达到阈值的小时数，再除以24形成暴露日。本项目缺少完全对应的共同逐小时样本，实际构造的是逐日Tmax超过连续阈值的温度超额积分，单位为°C-day。因此，项目只借用外部空间阈值产品，不声称移植、复现或数值近似公开逐小时暴露算法。

固定热干事件与该阈值暴露也不同：事件方向要求连续至少3天同时满足Tmax≥32°C和降水<1 mm；外生阈值暴露不要求降水条件或连续三天。两者只能在新的共同V3研究中作为独立暴露表征进行预设比较，不能合并成一个事件变量。
