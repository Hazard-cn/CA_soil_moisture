## 解释优先级

内部组合完整性96/100只说明三个override的机器结果、公共披露、审查链、测试和lineage在当时范围内一致；它不等于所有外部方法问题已经解决，也不把候选稿提升为95分投稿稿。网页端应优先检查研究问题和estimand是否一致，其次检查共同支持、固定效应、空间推断和多重检验，最后才讨论期刊定位。

## 推荐的下一步优先级

1. 重新冻结事件方向的两部分模型，区分事件发生、事件阳性年份累计事件天数和事件阳性年份强度；现有结果保留为历史总暴露规格。
2. 在同一V3事件数据家族内设计先行SM状态模块；执行前必须冻结无事件年份的可比状态定义和一季多事件的唯一聚合规则，产量方程保持每个`grid_id year`一行。该模块当前仅为新canonical建议。
3. 重新组织G185的低、高SR端点斜率和province×year结果，决定其能否形成中性条件斜率稿，或降为季节指数基准。
4. 阈值当前方向保持失败；只有形成外部阈值可迁移性或暴露测量误差的新研究问题并获得相应数据后再启动新版本。
