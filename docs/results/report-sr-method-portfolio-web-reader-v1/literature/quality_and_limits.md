# 文献证据质量与使用限制

37篇书目均通过来源存在性及DOI—题名匹配核验，但该结论不等于37篇全文均已阅读。自动请求仅1篇得到可读出版社页面内容，36篇受到验证码、403或跳转限制；其中4篇另做浏览器强化。逐篇学习点来自已保存的候选检索记录、权威索引摘要和可获得页面的综合，不应被表述为全文逐句复核。

代码可用性是另一个独立层级。16篇存在本地静态可读代码，共271个文件；全部未执行。其余21篇分别属于仅发现数据、未发现公开代码、官方链接受阻或需要向作者申请。静态可读不能证明依赖可满足、输入相符、数值可复现或算法可直接移植。

公开材料没有复制第三方全文、数据或许可证不明文件；仅复制8篇许可证明确材料的纯文本代码，并保留来源、许可证、固定版本和逐文件SHA-256。若后续方案依赖某篇论文的精确算法，必须重新核对论文正文、官方代码或数据仓库，并对本项目实现进行数值一致性测试；当前索引和许可代码集合仍只能用于提出、筛选和静态检查方法模块。
