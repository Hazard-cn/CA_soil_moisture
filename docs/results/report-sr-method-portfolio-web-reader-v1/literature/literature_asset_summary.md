## 公开发布边界

公开目录只收入书目信息、方法学习点、来源核验边界和代码状态摘要，不收入第三方代码、数据、归档、原始机器清单、Zotero内部标识或本机路径。该边界既避免许可证不明材料进入仓库，也防止网页端把静态代码集合误当成已经运行的复现环境。
