## 最有潜力、且能保持单一V3家族的下一步

优先考虑事件发生—严重程度两部分模型、先行SM状态依赖、固定事件下的下尾风险、留一区可迁移性和不设恢复阈值的SM动态轨迹。每个方案都必须冻结一个主estimand，并明确五区、物候、共同SR端点、空间推断、多重检验和STOP规则。
