## 使用边界

- `blocked`仅表示自动访问官方端点受阻，不能推断代码不存在。
- `author_request_only`仅表示已确认作者申请路径，不能写成代码已取得。
- `data_only`不能被改写为可执行代码包。
- 代码状态不能替代论文全文核验；逐篇方法主张仍应回到DOI、出版社页面或已取得全文。
