# Zotero 文献驱动的候选故事线综合计划

## 目标

仅使用当前 Zotero 文献，以 `output/figures/f4_b067_v2/figure_captions.md` 所示实证维度为结构约束，提出并排序六条面向综合性气候—农业子刊的候选故事线，最终推荐一条总主线与两条支撑主线。

## 执行步骤

1. 从 Zotero 选取 15–20 篇核心论文，覆盖气候条件性、干旱与高温、土壤水分、灌溉、联合胁迫、秸秆/残茬管理和区域适配。
2. 读取可用摘要或正文索引，核对论文实际结论、适用范围、处理定义和反向证据。
3. 建立“文献命题—预期经验模式—现有图组”矩阵，并区分直接证据、机制支持和边界证据。
4. 对六条候选主线分别形成中心命题、证据链、预期模式、图组映射、英文结论句、反论点、边界、补充检验和风险评级。
5. 依据新颖性、文献强度、实证贴合度和过度解释风险，推荐一条总主线与两条支撑主线。
6. 将结果写入 `quality_reports/2026-06-11_candidate_storylines_from_zotero.md`，并核验 DOI、Zotero item key、citation key、引用覆盖和 UTF-8 完整性。

## 约束

- 不新增文献检索或 Zotero 导入。
- 不预设地域排序或具体显著性结论。
- 区分秸秆还田、残茬覆盖、免耕和完整保护性农业。
- 使用 `conditional association`、`state-dependent buffering` 等非因果表述。
- 每条主线至少使用三篇独立文献，并纳入至少一项相反或限定性证据。
- 每条主线最多提出两个补充检验。
